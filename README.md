# Adarsha H — Portfolio

Personal portfolio site built with React + Vite.

## Local development
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```

## Deploy
See DEPLOY.md for step-by-step Vercel deployment instructions.

## Adding your resume
Place your resume PDF in the `public` folder and name it `resume.pdf`.
The "Download Resume" buttons in the nav and hero link to `/resume.pdf`.
